"An Info Ride Across the Bay" - A Visual Analytics system for analyzing and predicting the San Francisco Bay Area bike sharing network

Previous Next

This system presents 2 views - Explore and Predict. The explore view lets users analyze the Bay Area Bike Sharing network with the help of an interactive network visualization, an overview bar chart for all stations, individual station level statistics and much more. The visualizations support burshing-and-linking and filtering based on various parameters including date ranges, weather conditions, city level thresholds for selectively seeing stations in each city. The Predict view uses decision trees to predict hourly usage of bikes for different cities in the San Francisco Bay area given any climatic conditions.
You can try the system out at http://dva-bike-share.herokuapp.com/ (to begin, please select a date range and click apply filters - the initial population of views might take upto 20 seconds)

  • Developed For:

    CSE 6242 - Data and Visual Analytics

  • Date:

    Spring 2015

  • Keywords:

    d3.js, Python, Flask

FHIR DART - A system to visually create and analyze patient cohorts, compare medical providers and visualize careflow processes

Previous Next

A population health analysis system that allows creation of patient cohorts using filters and then facilitates further filtering using dynamic querying. Visualizations are provided highlighting common drugs presribed, lab tests ordered, consults ordered and careflow paths followed. The system also allows the user to select multiple providers and compare them based on aggregated values across all their encounters. The system was highly appreciated by Geisinger Health System and the course faculty. The data for the project was provided by the Georgia Tech and was compliant the FHIR standards.

  • Developed For:

    CS 6440 - Intro to Health Informatics

  • Date:

    Spring 2015

  • Keywords:

    d3.js, Java, Python

Genedive - A Visualization system for comparing genomes

A tool which allows comparison of mitochondrial genome data for various species of fishes, reptiles and mammals. Visualization included a phylogenetic tree, radial bar chart with links, comparison color maps and sequential exploration of genes with pan and zoom features. The tool also has an in-built wiki which allows reading about the visualizations and the species being explored. Working prototype can be found here.
This project is being developed in association with the Georgia Aquarium under the advise of Dr. Alistair Dove.
